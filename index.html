TabMind Privacy Policy

At TabMind, we take your privacy seriously. This privacy policy describes how we collect, use, and protect your information when you use our browser extension.

1. Data Collection

TabMind collects the following information:

  • Tab URLs and titles (for tab management functionality)
  • User preferences (saved locally in your browser)
  • Anonymous usage statistics
  • Tab context data for AI-powered features

Important: All data is stored locally on your device. We do not collect or store any personal information on our servers.

2. Data Usage

The collected data is used for:

  • Organizing and grouping your tabs
  • Providing AI-powered suggestions
  • Improving user experience
  • Context analysis for smart tab grouping

3. Data Storage

  • All data is stored locally in your browser using chrome.storage
  • No personal data is transmitted to external servers
  • Mellowtel integration is optional and requires explicit opt-in
  • Data persistence is limited to your browser environment

4. Security Measures

  • Secure HTTPS communication
  • Strict Content Security Policy (CSP) implementation
  • Limited permissions model
  • Service Worker isolation for enhanced security

5. User Rights

You have the right to:

  • Opt-in/out of AI features at any time
  • Control your saved data
  • Delete all stored data
  • Configure privacy settings

6. Mellowtel Integration

TabMind uses the Mellowtel library for AI features. This integration:

  • Is completely optional
  • Requires explicit user consent
  • Can be disabled at any time
  • Only processes anonymized data
